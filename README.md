# ImageRecognition

## Описание проекта

ImageRecognition — это декстопное приложение на WPF, разработанное на языке C#, предназначенное для классификации изображений рукописных цифр и простых геометрических фигур.

В основе системы лежит алгоритм k-ближайших соседей (k-Nearest Neighbors, kNN). Для ускорения поиска ближайших соседей реализована собственная структура данных — k-мерное дерево (KD-Tree).

Проект выполнен в рамках курсовой работы по дисциплине «Алгоритмы и структуры данных» и ориентирован на реализацию алгоритмов без использования готовых библиотек машинного обучения.

## Основные возможности

- Собственная реализация структуры данных KD-Tree
- Реализация алгоритма k-ближайших соседей (kNN)
- Предобработка изображений и извлечение признаков
- Хранение данных в PostgreSQL
- Измерение производительности алгоритмов
- Вычисление точности классификации
- Сравнение линейного поиска и поиска с использованием KD-Tree
- Графический интерфейс на WPF

## Используемые технологии

- C#
- .NET
- WPF
- PostgreSQL
- Npgsql
- Архитектурный паттерн MVVM

## Архитектура системы

Приложение построено по многоуровневой архитектуре:

- Presentation Layer — пользовательский интерфейс (WPF)
- Application Layer — прикладная логика и сценарии использования
- Domain Layer — алгоритмы и структуры данных
- Infrastructure Layer — взаимодействие с базой данных PostgreSQL

Алгоритмы KD-Tree и kNN реализованы полностью самостоятельно без использования библиотек машинного обучения (ML.NET, scikit-learn и других).

## Конвейер обработки изображения

1. Загрузка изображения
2. Приведение к фиксированному размеру (например, 16x16)
3. Преобразование в оттенки серого
4. Формирование вектора признаков (например, 256 признаков)
5. Сохранение признаков в базе данных
6. Построение KD-Tree
7. Классификация методом kNN

## Структура базы данных

База данных PostgreSQL содержит следующие таблицы:

- classes — классы изображений
- images — информация об изображениях
- features — векторы признаков
- models — параметры модели (значение k)
- experiments — результаты экспериментов
- predictions — результаты классификации

Векторы признаков хранятся в формате массива FLOAT8[].

## Реализованные алгоритмы

- Построение k-мерного дерева
- Поиск ближайшего соседа в KD-Tree
- Поиск k ближайших соседей
- Евклидово расстояние
- Классификация методом большинства голосов
- Вычисление точности (Accuracy)
- Измерение времени выполнения операций

## Оценка производительности

В рамках проекта проводится анализ:

- Времени построения KD-Tree
- Среднего времени поиска ближайших соседей
- Точности классификации
- Сравнения линейного поиска и поиска с использованием KD-Tree

## Структура решения

Проект разделён на следующие модули:

- ImageRecognition.UI — пользовательский интерфейс
- ImageRecognition.Application — прикладная логика
- ImageRecognition.Domain — алгоритмы и структуры данных
- ImageRecognition.Infrastructure — работа с PostgreSQL
- ImageRecognition.Tests — модуль тестирования

## Назначение проекта

Цель проекта — разработка и исследование алгоритма классификации изображений на основе метода k-ближайших соседей с использованием k-мерного дерева и проведение анализа эффективности алгоритмических решений.
